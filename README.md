# Airtable Challenge

**Requirements**
```
node >=10.14.2
yarn >=1.15.2
```

**Commands**
```bash
# install dependencies
yarn install

# start dev server
yarn dev
```

* How long you spent on the assignment.
  > ~7 hours
* What you like about your implementation.
  > I love using redux + recompose to separate business logic from component. I like my solution for text overflow (ellipsis and reveal on hover). I think figuring out the span logic was the most challenging part.
* What you would change if you were going to do it again.
  > Not much, I would just keep building on it. I might go for a mobile view (vertical scroll, similar to google calendar).
* How you made your design decisions. For example, if you looked at other timelines for inspiration, please note that.
  > I looked at the Airtable gant timeline, google calendar, and this image for inspiration: https://tinyurl.com/y243hn6z
